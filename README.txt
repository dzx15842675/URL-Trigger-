一、程序概述

URL Trigger是一款轻量级局域网远程控制工具，支持通过浏览器访问指定URL，实现远程触发电脑打开指定程序的功能。程序运行后常驻系统托盘，操作简单便捷，还可配合智能设备或NFC标签实现自动化操作，满足用户多样化的远程控制需求。

二、快速上手指南

（一）程序启动

找到URL Trigger程序安装目录，双击主程序文件（URLTrigger.exe），程序将自动后台运行，桌面无弹窗显示。

程序启动后，电脑右下角系统托盘区域会出现URL Trigger图标，表明程序已正常运行。

（二）基础远程控制操作

确定访问地址：

若在同一台电脑上操作，直接使用默认地址：http://127.0.0.1:5000/open

若使用手机或其他局域网设备操作，需将地址中的127.0.0.1替换为运行URL Trigger程序的电脑IP地址。查看电脑IP地址方法：Windows系统可通过cmd命令行输入ipconfig查看，Mac系统可通过系统偏好设置-网络查看。

触发程序打开：在浏览器地址栏输入上述正确地址并访问，默认将触发打开电脑自带的记事本程序。

（三）后台管理设置

进入后台管理页面：

本地访问地址：http://127.0.0.1:5000/admin

局域网设备访问地址：将127.0.0.1替换为服务端电脑IP地址。

登录验证：

输入账号：admin

输入密码：admin123

若忘记密码，可使用备用密码：Administrator

自定义设置：登录后台后，可修改默认触发打开的程序，根据页面提示选择电脑中已安装的应用程序路径，保存设置后，再次访问/open地址将打开自定义的程序。

（四）程序退出

右键点击系统托盘中的URL Trigger图标，在弹出的菜单中选择“退出”选项，即可关闭程序。

三、扩展功能说明

（一）配合小爱同学实现自动化

确保小爱同学设备与运行URL Trigger的电脑处于同一局域网。

在小爱同学APP中创建自定义指令，例如设置“打开办公软件”指令，触发动作访问URL Trigger的/open地址（需替换为电脑实际IP）。

完成设置后，通过语音唤醒小爱同学并说出指令，即可远程触发电脑打开指定程序。

（二）配合NFC标签实现自动化

使用NFC标签写入工具，将URL Trigger的/open地址（替换为电脑实际IP）写入NFC标签。

将NFC标签贴在常用位置，如桌面、手机壳背面，使用支持NFC功能的设备触碰标签，即可快速触发电脑打开指定程序。

问题：浏览器访问地址后无反应？
